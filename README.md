# @theroyalwhee0/mailtpl


## What is this?
A mail template library. Build templates in HTML and CSS and compile into HTML with inline styles.


## Installation
`npm install @theroyalwhee0/mailtpl`  
*or*  
`yarn add @theroyalwhee0/mailtpl`
